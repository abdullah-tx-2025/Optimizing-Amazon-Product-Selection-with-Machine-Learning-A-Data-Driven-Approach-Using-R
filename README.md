# Optimizing-Amazon-Product-Selection-with-Machine-Learning-A-Data-Driven-Approach-Using-R
Launching an online store on Amazon comes with uncertainty. Sellers often rely on trends, intuition, or anecdotal recommendations. However, without a structured approach or access to historical sales data, this guesswork leads to poor inventory decisions, unsold stock, and lower search rankings. To overcome this challenge we used ML models.
